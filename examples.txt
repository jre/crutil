Note: first you will need to change into the crutils folder:
cd crutil

To update crutils:
git pull

Initial setup:
./setup.sh

Build database from chain and cr guru:
./cr-import.py -g

List raiders:
./cr-report.py

Show raider gear:
./cr-report.py -r <raider-id>

After reloading the raiders page, update local database andreport best gear:
./cr-import.py -r <raider-id>; ./cr-report.py -b -r <raider-id>
