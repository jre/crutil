Note: first you will need to change into the crutils folder:
cd crutil

To update crutils:
git pull

Initial setup:
./setup.sh

Build database from chain and cr guru:
./cr-import.py -g

List raiders:
./cr-report.py

Show raider gear:
./cr-report.py -r <raider-id>

Pull spreadsheet data and report best gear:
./cr-import-sheet.py; ./cr-report.py -b -r <raider-id>
